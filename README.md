

# **Guia de Inicialização do Projeto**

## **1. Clonando e executando o projeto localmente**

```sh
# Clone o repositório
git clone <SEU_GIT_URL>

# Acesse o diretório do projeto
cd <NOME_DO_PROJETO>

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev

#OBS : tem que ter o NODE.JS na maquina 
```

---

## **2. Edição de arquivos diretamente no GitHub**

1. Acesse o arquivo desejado no repositório.
2. Clique no ícone de lápis (“Edit”).
3. Realize as alterações necessárias.
4. Confirme o commit para registrar a mudança.

---

## **3. Edição com GitHub Codespaces**

1. Abra a página principal do repositório.
2. Clique em **Code** (botão verde).
3. Selecione a aba **Codespaces**.
4. Clique em **New codespace** para abrir um ambiente pronto para desenvolvimento.
5. Edite, commite e faça push das alterações diretamente pelo Codespace.

---

## **4. Stack Tecnológica**

O projeto está estruturado sobre uma arquitetura moderna e de alta performance:

* **Vite**
* **TypeScript**
* **React**
* **shadcn-ui**
* **Tailwind CSS**

Essa combinação garante velocidade, escalabilidade e fluidez no ciclo de desenvolvimento.

---

## **5. Conectando um domínio**

1. Acesse **Project → Settings → Domains**.
2. Clique em **Connect Domain**.
3. Siga o fluxo de configuração apresentado na plataforma.
